# proyectoAssitant

CLI wrapper interactivo para [opencode](https://opencode.ai). Envía mensajes a opencode y muestra la respuesta en un bucle hasta que escribes `adios`.

## Requisitos

- Python 3.x
- [opencode](https://opencode.ai) instalado y accesible en `PATH`

## Uso

```bash
python main.py
```

Escribe tu mensaje y presiona Enter. Para salir, escribe `adios`.
